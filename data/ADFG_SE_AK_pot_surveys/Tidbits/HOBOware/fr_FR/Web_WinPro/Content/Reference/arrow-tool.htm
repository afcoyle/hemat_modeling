L'outil flèche

L'outil flèche se sélectionne de deux manières :

  • Cliquez sur l'icône Flèche de la barre d'outils ; ou
  • Appuyez sur la touche H du clavier.

Astuce : Si vous utilisez un autre outil et ne voulez pas en changer, vous pouvez appuyer sur la touche N (suivant) de votre clavier pour sélectionner chaque élément du graphe (y compris les éléments masqués, tels que les axes de valeur pour les états ou les événements) dans l'ordre.

L'article sélectionné (s'il est visible) est grisé ou encadré. Vous pouvez ensuite cliquer à droite sur l'article sélectionné, cliquer deux fois dessus ou appuyer sur la touche Entrée de votre clavier pour ouvrir la fenêtre Propriétés correspondante.

Pour désélectionner, cliquez sur une zone au sein des axes qui n'ait pas de données de série ; cliquez sur le menu Affichage et choisissez Désélectionner tout ; ou appuyez sur la touche D de votre clavier.
