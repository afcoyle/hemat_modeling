HOBOware Setup Assistant (HOBOware Pro Only)

When you open HOBOware Pro for the first time, you are greeted by the HOBOware Setup Assistant, a series of dialogs intended to help you quickly change some key preference settings for device types and serial ports, units of measurement, and data assistants. To run the Setup Assistant, click Start and follow the prompts to change some key preferences, listed below.

NOTE: The Setup Assistant is optional. If you click Cancel, you will be able to use HOBOware Pro with its default settings, as listed below.

To run the Setup Assistant at any time, go to:

Windows:  File > Preferences > General

Mac: HOBOware > Preferences > General

and click the Setup Assistant button in the bottom right of the pane.

Setup Preferences

Device Types

Serial Port(s) to use:  

  • Support serial devices on COM1 (Windows) 
  • Support serial devices on the default serial port only (Mac)

The default is USB. If you enable Serial Devices you can select the port to use.

Unit System

  • US (default)
  • SI

Data Assistants

Select the installed Data Assistants you want to show when relevant data is present. All loaded Data Assistants are checked by default. You can also load a new Data Assistant.
