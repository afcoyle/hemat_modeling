The Device Menu

  1. Launch

Select this option to access the Launch window to configure launch parameters and launch a logger.

  1. Readout

Select this option to copy data from a logger to your computer, allowing you to save the data in a datafile and view the plot in HOBOware.

  1. Status

Select this option to view real-time information about the attached logger, including: state of the logger, time logging will begin, current readings, logger memory used.

  1. Stop

Select this option to stop a device.

  1. Manage Shuttle (HOBOware Pro Only)

Select this option to manage a HOBO shuttle. See Working with a Shuttle for more information.

  1. Configure Modules/Ports (HOBOware Pro Only)

Select this option to configure analog sensor ports or Flexsmart modules.

  1. Control Relay

Select this option to configure the default state of a relay on a HOBO U30, or to activate or deactivate the relay for testing. See Controlling the Relay on the HOBO U30 Station for more information.

  1. Select Device

Select this option to select a device that is connected to the computer.

  1. Default Action (HOBOware Pro Only)

Select this option to configure HOBOware Pro to perform the same action (Launch, Readout, or get Status) every time a logger is connected. See Repeating an Action on Multiple U-Series Devices for more information.
