Enregistrement de fichiers à partir d'une navette

Avant de pouvoir enregistrer des fichiers, vous devez les délester de la navette. Voir « Délestage de fichiers d'une navette ».

Enregistrement de fichiers

L'illustration ci-dessous montre les parties de la boîte de dialogue Gestion de la navette liée à l'enregistrement de fichiers.

1. Dossier d'enregistrement

L'emplacement d'enregistrement par défaut est indiqué dans le champ Enregistrer dossier. Pour choisir un autre emplacement, cliquez sur Choisir et naviguez jusqu'à l'emplacement.

2. Ouverture automatique des fichiers

Pour que le dossier s'ouvre automatiquement sur le bureau une fois les fichiers enregistrés, cochez :

Windows : Ouvrir le dossier dans l'Explorateur Windows après Enregistrer.

Mac : Ouvrir le dossier dans le Finder après Enregistrer.

3. Choisissez les fichiers à enregistrer.

Par défaut, tous les fichiers que vous délestez sont sélectionnés pour enregistrement.

Pour désélectionner des fichiers individuels, cochez la case à côté de chaque fichier.

Pour désélectionner tous les fichiers, cliquez sur le bouton Désélectionner tous.

Pour enregistrer tous les fichiers (si vous en avez déjà désélectionné quelques-uns), cliquez sur le bouton Sélectionner tous.

4. Nom du fichier de données

Entrez un nouveau nom au besoin.

5. Annuler enregistrement

Pour quitter sans enregistrer le moindre fichier, cliquez sur Annuler enregistrement.

6. Enregistrer sélectionnés

Cliquez sur ce bouton pour enregistrer tous les fichiers cochés.

Les fichiers sont enregistrés dans le dossier spécifié.
