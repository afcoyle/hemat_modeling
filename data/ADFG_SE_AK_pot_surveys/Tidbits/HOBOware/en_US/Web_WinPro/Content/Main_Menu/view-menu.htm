The View Menu

 

  1. Zoom In

Select this option to access the Zoom In tool.

  1. Zoom Out

Select this option to access the Zoom Out tool.

  1. View Full Scale

Select this option to view the graph in full scale.

  1. Details Pane

Select this option to show or hide the Details Pane.

  1. Points Table

Select this option to show or hide the Points Table.

  1. Show Title

Select this option to show or hide the Graph Title.

  1. Show Legend

Select this option to show or hide the Legend.

  1. Mark All Points

Select this option to mark all points.

  1. Vertical Gridlines

Select this option to show or hide vertical gridlines on the graph. See Displaying Gridlines in Customizing a Plot for more information.

  1. Horizontal Gridlines

Select this option to show or hide horizontal gridlines on the graph. See Displaying Gridlines in Customizing a Plot for more information.

  1. Select None

Select this option to deselect all elements in the graph.

  1. Remove Crosshair

Select this option to remove crosshairs from the graph.

  1. Remove Subset Stats

Select this option to remove a subset from a graph.

  1. Bring Series To Front

Select this option to bring a series to the front in a graph. See Changing the Order of a Series in Customizing a Plot for more information.

  1. Send Series To Back

Select this option to send a series to the back of a graph. See Changing the Order of a Series in Customizing a Plot for more information.
