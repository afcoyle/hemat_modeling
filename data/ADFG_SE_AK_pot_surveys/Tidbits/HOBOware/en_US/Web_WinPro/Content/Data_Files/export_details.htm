Exporting Details

Overview

You can export the information in the Details pane to a text file, which can be opened in any text editor, or imported into many other types of programs (word processors, spreadsheets, etc.).

To configure HOBOware to automatically export the data to a text file every time the logger is read out, see Export Settings.

Steps

To export details:

  1. Make sure the Details pane includes the series you want to export. This might include series pasted from other datafiles, or series derived through filters or data assistants.
  2. From the File menu, select Export Details.
  3. Accept the default file name, or type a new name.
  4. Click Save.
