Boîte de dialogue Sélectionner le canal à configurer

  1. Cliquez sur le signe plus à côté du nom du module/capteur pour voir les canaux disponibles.
  2. Sélectionnez le canal de votre choix.
  3. Cliquez sur le bouton correspondant à l'action que vous voulez exécuter : Désactiver, Configurer, Rafraîchir.

Si vous cliquez sur Configurer, la boîte de dialogue Configurer canal s'affiche.
