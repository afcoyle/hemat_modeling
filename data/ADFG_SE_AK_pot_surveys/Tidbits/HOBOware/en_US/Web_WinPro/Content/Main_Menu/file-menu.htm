The File Menu

From the File Menu you can open datafiles and projects, import and export data, and print plot elements.

  1. Open Data File

Select this option to ppen a compatible datafile. See Opening Files for more information.

  1. Open Project

Select this option to open a HOBOware project. See Opening Files for more information.

  1. Recent Files

Select this option to quickly access the last 10 files you opened.

  1. Close

Select this option to close the active file or project.

  1. Close All

Select this option to close all open files and projects.

  1. Save Datafile

Select this option to save the active datafile.

  1. Save Project

Select this option to save the active project.

  1. Import Text Data (HOBOware Pro Only)

Select this option to graph non-HOBOware data by importing data from certain text files. See Importing Text Files for more information.

  1. Export Details

Select this option to export the information in the Details Pane into a text file that can be opened in any text editor, or imported into many other types of programs (word processors, spreadsheets, etc.). See Exporting Details for more information.

  1. Export Points as Excel Text

Select this option to export the points shown in the Points table to a text file (.csv or .txt) for importing into Microsoft Excel and other applications. See Exporting Points as Excel Text for more information.

  1. Page Setup

Select this option to set up your paper size, margins, and orientation for printing a graph.

  1. Print Preview

Select this option to preview the page for printing a graph.

  1. Print

Select this option to print a graph.

  1. Print Details

Select this option to print the Details Pane.

  1. Print Points

Select this option to print the Points Table.

  1. Preferences

Select this option to access the Preferences window to change the default behavior of HOBOware. See Setting Preferences for more information.

  1. Quit

Select this option to close HOBOware.
