Conversion d’unités

Utilisez la boîte de dialogue Convertir tracé pour modifier les unités de mesure de séries multiples dans un tracé (s’il y a lieu).

  1. Pour chaque série prenant en charge les options d’unité, sélectionnez les unités souhaitées sur la liste déroulante.
  2. Cliquez sur Convertir.

Les unités affichées sont modifiées sur l'axe, dans la légende, dans le volet Détails et dans le tableau Points.
