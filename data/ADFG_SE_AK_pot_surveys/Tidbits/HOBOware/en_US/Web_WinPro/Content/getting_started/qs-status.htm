Checking Logger Status

  1. Click the Status icon on the toolbar.

The Status Window appears (see page 17).

Watch as the Current Readings update in real time.

  1. Change Current State

If you are using a U Series logger that has a button, hold the button down for several seconds.

Watch the Current States field in the Status window change to Button Down.

Release the button.

  1. Change Sensor Readings.

If you are measuring temperature, change the temperature reading by holding the logger or sensor in your hands for a minute or two. Watch the Current Readings in the Status window change as the conditions change.

  1. Close the Status window by clicking OK.
