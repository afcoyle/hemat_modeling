Exporting Data in BoxCar Format

The Export Settings pane gives you flexibility in setting various date and time formats, allowing you to export and import data in a format compatible with Onset's Box Car Pro software.

Select the Use default Boxcar Pro export settings option to have all Export settings set to Box Car defaults. See Export Settings for more information.

You can also change the following Display Preferences:

  • 12 hour and 24 hour time formats
  • MDY, DMY formats
  • Date separator
