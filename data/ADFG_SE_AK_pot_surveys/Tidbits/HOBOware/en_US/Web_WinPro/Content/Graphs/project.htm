Overview of the HOBOware Interface

The illustration below shows the HOBOware Main Window, showing the plot and other elements that are shown when you generate a plot or open an existing data file.

  1. Main Menu

From the Main Menu you can perform operations on a device or a project.

  1. Toolbar

The toolbar includes icons for performing operations on a device as well as tool to manipulate a plot or graph. For more information see The Toolbar.

  1. Points Table

Logger data presented in a tabular format. See The Points Table.

  1. Details Pane

Information and statistics about the launch.

  1. Plot

A graphical representation of the sensor readings and events.

  1. Views

Use these icons to toggle between views when you have multiple plots open.

  1. Message Bar
