Checking Logger Status

 

  1. On the Status icon on the toolbar to open the Status window.
  2. Watch as the Current Readings update in real time.
  3. Change Current State

If you are using a U-Series logger that has a button for logging events, press the button for several seconds and watch the Current States in the Status window change to Button Down. Release the button.

  1. Change Sensor Readings

Change the sensor readings by changing the conditions. For example, if you are measuring temperature, change the temperature reading by holding the logger or sensor in your hands for a minute or two. Watch the Current Readings change as the conditions change.

  1. Close the Status window by clicking OK.
