The Launch Window for U-Series Loggers

The Launch Window for all U-Series loggers (except the U30 Station) is shown below.

See Launch Parameters for U-Series Loggers for more information.

  1. Logger Information: Logger Type, Serial Number, Deployment Number, Battery or Battery Level.
  2. Description: Enter a description for the launch, up to 40 characters, in this field. This description is used as the default file name when you read out and save the data recorded by the logger. It is also the title of the graph
  3. Channels to Log: See Channels to Log Guidelines for Station Loggers.
  1. Logging Interval: The rate at which you want the logger to record data.
  2. Logging Duration: Approximate time it would take to fill the logger memory based on the logging interval and currently enabled sensors.
  3. Launch Options: When to Launch the logger.
  4. Help Button: Click to open the on-line Help for the Launch window.
  5. Skip launch window next time (HOBOware Pro only): Check this box if you would like to bypass this window the next time you launch a device.
  6. Launch Button: Click to launch the logger.
  1. Utilities (not available on all loggers): Includes Scaling, Alarms.
